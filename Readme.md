## Project setup
```
npm install
```

### Compiles for development
```
npm start
```

### Prerequisites
* [NodeJS](https://nodejs.org/en/download/) 
* NPM (should be bundled with NodeJS)
* [MongoDB](https://www.mongodb.com/)
* [Postman](https://www.postman.com/)
